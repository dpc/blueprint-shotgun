# Blueprint Shotgun Boosted

Blueprint Shotgun Boosted is a fork of [_CodeGreen's Blueprint Shotgun](https://mods.factorio.com/mod/blueprint-shotgun?from=search).

This fork makes the shotgun more useful by introducing more upgrades that improve its performance through the early and mid game, reducing tedious manual item placement and keeping it useful even when construction bots are already an alternative. It keeps the original in-game prototype, item, technology, setting, and control names for compatibility with existing saves and integrations; the fork is renamed only at the mod/package level. Factorio is also told that this fork is incompatible with the original `blueprint-shotgun` mod, so both are not enabled together.

Credits for the original mod, concept, code, graphics, and sounds belong to the Blueprint Shotgun authors. This fork exists as a balance/progression tweak for players who want a stronger early-to-mid-game alternative to manual placement.
